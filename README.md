## ◈ Sobre

Este repositório documenta minha evolução na disciplina **ECM252 — Linguagens de Programação II**, uma jornada completa pelo desenvolvimento web moderno: do HTML/CSS até Angular, React, aplicações mobile com Ionic, backend com Node.js e PHP, containerização com Docker e fundamentos de IHC.

Uma das disciplinas mais abrangentes do curso — cobre front-end, back-end, mobile e UX num único semestre.
